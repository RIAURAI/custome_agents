Connect Jira

Link your Atlassian Jira account

Your Jira URL looks like: https://yourteam.atlassian.net
Find it in your browser address bar when logged into Jira.

Go to Atlassian API Tokens → Create API token → Give it a label → Copy the token.
Important: You can only see the token once! Save it now.

Use the email associated with your Atlassian account. This is used together with the API token for authentication (Basic Auth).

If you want to link a specific project, find the Project Key (e.g. WH, DEV) in Jira → Project Settings → Details.
Leave empty to access all projects.

Fill in your Site URL, Email, and API Token. Click Connect & Save — we'll verify by calling the Jira API.
