Connect Discord

Set up your Discord bot step by step

Go to Discord Developer Portal → Click New Application → Give it a name → Create.

In your App → Bot tab → Click Reset Token → Copy the token.
Enable these Privileged Intents:
• MESSAGE CONTENT INTENT
• SERVER MEMBERS INTENT
• PRESENCE INTENT

OAuth2 → URL Generator → Select scopes: bot, applications.commands
Bot Permissions: Send Messages, Read Message History, Manage Messages
Open the generated URL → Select your server → Authorize.

Enable Developer Mode: User Settings → Advanced → Developer Mode ON.
Right-click your server name → Copy Server ID. This is your Guild ID.

Fill in your Bot Token and Guild ID below. Webhook URL is optional (for receiving events). Click Connect & Save — we'll verify the token.
