Connect Slack Workspace

Each user connects their own workspace — fully dynamic, nothing hardcoded

  1. Go to api.slack.com/apps → Create New App → From scratch.
  2. Under OAuth & Permissions → Bot Token Scopes — add:
    channels:read channels:history channels:join chat:write users:read groups:read groups:history im:read im:history im:write mpim:read mpim:history
  3. Click Install to Workspace → authorize → copy the Bot User OAuth Token (xoxb-…).
  4. Under Basic Information → copy the Signing Secret.
  5. (For real-time Socket Mode bot) Under Basic Information → App-Level Tokens → create a token with scope connections:write → copy the App Token (xapp-…).
  6. Enable Socket Mode (Basic Information → Socket Mode toggle).
  7. Under Event Subscriptions → Subscribe to bot events: message.channels, message.groups, message.im, app_mention.
  8. Paste all tokens below and click Validate & Connect. Done!
