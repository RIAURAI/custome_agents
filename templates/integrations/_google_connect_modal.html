Connect Google Workspace

Follow the steps below to enable Gmail, Drive, Docs, Sheets, Forms, Calendar & Meet

  1. Open console.cloud.google.com
  2. Click the project selector at the top → New Project
  3. Enter a Project Name (e.g. WorkHub Integration) → Click Create
  4. Make sure the new project is selected in the top bar before continuing
One Google Cloud project per company is recommended.

Go to APIs & Services → Library and enable each API below:

Click each card to open the API in Google Cloud Console, then click Enable.

  1. Go to APIs & Services → OAuth consent screen
  2. Choose User Type:
    • Internal — only for Google Workspace org users (recommended for businesses)
    • External — any Google account (requires Google verification for production)
  3. Fill in App name (e.g. WorkHub), User support email, Developer contact email
  4. Under Scopes → click Add or Remove Scopes → add the scopes listed in Step 5 below
  5. Click Save and Continue through all sections
For External apps in testing mode, add your email addresses as Test users.

  1. Go to APIs & Services → Credentials
  2. Click + Create Credentials → OAuth client ID
  3. Application type → choose Web application
  4. Name it (e.g. WorkHub Web Client)
  5. Under Authorized redirect URIs — click + Add URI and paste:
    {{ google_redirect_uri }}
  6. Click Create
  7. A dialog shows your Client ID and Client Secret — copy both
The Client Secret is shown only once. Copy it immediately or download the JSON file.

Connect Google Workspace

Enter your Google Cloud OAuth credentials — each company connects its own Google Cloud project

  1. Go to console.cloud.google.com → Select or create a project.
  2. Navigate to APIs & Services → OAuth consent screen → Configure (External or Internal for Workspace orgs).
  3. Go to APIs & Services → Credentials → Click Create Credentials → OAuth Client ID.
  4. Choose Web application as Application type.
  5. Under Authorized redirect URIs, add:
    {{ google_redirect_uri }}
  6. Enable the required APIs in APIs & Services → Library: Gmail API, Google Calendar API, Google Meet REST API, Google Drive API, Google Docs API, Google Sheets API, Google Forms API.
  7. Copy the Client ID and Client Secret from the credentials page.
  8. Paste them below and click Save & Connect. You'll be redirected to Google to authorize.
