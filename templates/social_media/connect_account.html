How to get your credentials
{% if platform == "whatsapp" %}
  1. Go to Meta Developer Portal
  2. Create or select your WhatsApp Business App
  3. Navigate to WhatsApp → API Setup
  4. Copy your Phone Number ID, WABA ID, and generate a Permanent Token
  5. Set your Webhook URL (shown below) to receive messages
{% elif platform == "linkedin" %}
  1. Go to LinkedIn Developer Portal
  2. Create an app → get your Client ID and Client Secret
  3. Request OAuth 2.0 scopes: r_liteprofile, w_member_social, r_organization_social
  4. Generate an Access Token via OAuth flow
{% elif platform == "facebook" %}
  1. Go to Meta Developer Portal
  2. Select your App → Add Facebook Login product
  3. Generate a Page Access Token with pages_messaging, pages_manage_posts permissions
  4. Copy your Page ID from Page Settings → Transparency
{% elif platform == "instagram" %}
  1. Go to Meta Developer Portal
  2. Set up Instagram Graph API on your app
  3. Link your Instagram Professional Account to a Facebook Page
  4. Generate token with instagram_basic, instagram_manage_messages permissions
  5. Copy your Instagram Business Account ID
{% else %}
  1. Go to Twitter Developer Portal
  2. Create a Project and App → get API Key and API Secret
  3. Generate Bearer Token for read access
  4. Generate Access Token & Secret for write access (posting, DMs)
{% endif %}
