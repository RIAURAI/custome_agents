Platform Setup Guide

Step-by-step instructions to connect your social media accounts

Dashboard
1
WhatsApp Business API
Connect via Meta Business Suite
  1. Go to Meta Business Suite
  2. Create or select your Business Account
  3. Navigate to WhatsApp Manager → Getting Started
  4. Add a phone number and verify it
  5. Generate a Permanent Access Token under System Users
  6. Copy the Phone Number ID and WABA ID
  7. Set webhook URL to:
    {{ request.scheme }}://{{ request.get_host }}/social/webhooks/whatsapp/
  8. Paste credentials in the Connect modal on the dashboard
2
Facebook Page
Connect a Facebook Page for posting and messaging
  1. Go to Facebook Developers
  2. Create a new App (Business type)
  3. Add the Pages product
  4. Generate a Page Access Token with pages_manage_posts, pages_messaging, and pages_read_engagement permissions
  5. Copy your Page ID
  6. Paste credentials in the Connect modal
3
Instagram Business
Requires a connected Facebook Page
  1. Switch your Instagram to a Business Account
  2. Link it to a Facebook Page
  3. Use the same Facebook App from step 2
  4. Add the Instagram Graph API product
  5. Generate a token with instagram_basic, instagram_content_publish, instagram_manage_messages permissions
  6. Your Instagram Business ID is your Account ID
4
LinkedIn
Post to your Company Page
  1. Go to LinkedIn Developers
  2. Create an App and verify it with your Company Page
  3. Request w_member_social and r_organization_social permissions
  4. Generate an OAuth 2.0 access token
  5. Paste your token and Organization ID in the Connect modal
5
Telegram Bot
Create a bot via @BotFather
  1. Open Telegram and message @BotFather
  2. Send /newbot and follow the prompts
  3. Copy the API Token (this is your API Key)
  4. Set the webhook URL:
    {{ request.scheme }}://{{ request.get_host }}/social/webhooks/telegram/
  5. To find your Chat ID, message the bot and check getUpdates
  6. Paste credentials in the Connect modal
