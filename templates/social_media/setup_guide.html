Setup Guide

Step-by-step instructions to connect your social media platforms.

Quick Start — 3 Steps
1
Connect Account

Go to Dashboard → Click "Connect" → Enter API credentials

2
Set Up AI Bot

Go to Bot Settings → Create auto-reply rules with custom AI instructions

3
Configure Webhook

Set your webhook URL in the platform's developer console

WhatsApp Business API
  1. Go to Meta for Developers and log in
  2. Create a new App → Select "Business" type
  3. Add "WhatsApp" product to your app
  4. Go to WhatsApp → Getting Started — copy your Phone Number ID and WABA ID
  5. For permanent token: Business Settings → System Users → Generate Token with whatsapp_business_messaging permission
  6. Set up Webhook:
    • URL: https://your-domain.com/social/webhooks/whatsapp/
    • Verify Token: WHATSAPP_VERIFY_TOKEN from your .env
    • Subscribe to: messages
What to enter in WorkHub
Account Name: Your business name
Access Token: System User permanent token
Phone Number ID: From Getting Started page
WABA ID: WhatsApp Business Account ID
ENV: WHATSAPP_VERIFY_TOKEN=your-secret-token
LinkedIn
  1. Go to LinkedIn Developers → Create App
  2. Associate with your LinkedIn Company Page
  3. Go to Auth tab → Copy Client ID & Client Secret
  4. Request access to: Share on LinkedIn, Sign In with OpenID Connect
  5. Generate Access Token via OAuth 2.0 (scopes: w_member_social, r_liteprofile)
What to enter in WorkHub
Account Name: Company/profile name
Access Token: OAuth 2.0 token
Account ID: Organization URN
Profile URL: LinkedIn page URL
Note: LinkedIn tokens expire in 60 days — refresh periodically.
Facebook Messenger
  1. Go to Meta for Developers → Create Business App
  2. Add "Messenger" product
  3. Select your Facebook Page → Generate Page Access Token
  4. Copy the Page ID
  5. Set up Webhook:
    • URL: https://your-domain.com/social/webhooks/facebook/
    • Subscribe to: messages, messaging_postbacks
What to enter in WorkHub
Account Name: Facebook Page name
Access Token: Page Access Token
Page ID: Facebook Page ID
ENV: FACEBOOK_VERIFY_TOKEN=your-secret-token
Instagram
  1. Switch Instagram to Business or Creator account
  2. Connect Instagram to your Facebook Page
  3. Use same Meta app → Add "Instagram" product
  4. Get Instagram Business Account ID (via Facebook Page Settings)
  5. Use the same Page Access Token as Facebook
What to enter in WorkHub
Account Name: Instagram handle
Access Token: Facebook Page token
Page ID: IG Business Account ID
Profile URL: instagram.com/handle
Twitter / X
  1. Go to Twitter Developer Portal
  2. Apply for Elevated or Pro access (needed for DMs)
  3. Create Project & App → Copy API Key, API Secret, Bearer Token
  4. Generate Access Token & Secret with read/write/DM permissions
  5. Set up Account Activity API → Register webhook: https://your-domain.com/social/webhooks/twitter/
What to enter in WorkHub
Account Name: Twitter handle
Access Token: Bearer Token
Account ID: Twitter User ID
ENV: TWITTER_CONSUMER_SECRET=your-consumer-secret
Required .env Settings
# Social Media Webhooks WHATSAPP_VERIFY_TOKEN=your-whatsapp-verify-token FACEBOOK_VERIFY_TOKEN=your-facebook-verify-token TWITTER_CONSUMER_SECRET=your-twitter-consumer-secret # AI Auto-Reply Bot OPENAI_API_KEY=sk-your-openai-api-key # Token Encryption FERNET_KEY=your-fernet-encryption-key
Local Testing with ngrok:
ngrok http 8000 # Use the https URL → e.g. https://abc123.ngrok.io/social/webhooks/whatsapp/
